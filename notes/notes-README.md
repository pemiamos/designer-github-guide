# 学习笔记

这个文件夹记录实际使用 GitHub 过程中的观察、思考与踩过的坑。

## 如何浏览所有笔记

由于 GitHub Pages 的限制,这里无法自动列出文件夹中的所有笔记。要浏览完整的笔记列表,请访问 GitHub 仓库页面:

👉 [在 GitHub 上查看所有笔记](https://github.com/pemiamos/designer-github-guide/tree/main/notes)

在 GitHub 仓库页面上,你会看到完整的文件列表,并可按最新修改时间排序。每个文件都是一篇独立的笔记,点击即可阅读。

## 关于这个现象

"为什么 GitHub Pages 上看不到文件列表"本身就是一个值得记录的发现——GitHub Pages 是真正的网站托管服务,它和 GitHub 仓库页面是两套不同的系统,对"没有 index 文件的文件夹"的处理方式完全不同。

详细的原因分析和解决方案,可以参考本文件夹中的:
